Disaster Text Classifier

A lightweight machine learning tool that estimates whether a short text or tweet describes a real-world disaster event. Built for rapid deployment in humanitarian and development contexts.

Limitations

This classifier is trained on short English disaster-related text and may be unreliable on sarcasm, ambiguous language, uncommon events, multilingual input, or rapidly evolving real-world situations. The confidence score is a model estimate, not verified situational awareness.

Endpoints
GET /health
Health-check endpoint. Returns 503 if the model is not loaded, otherwise 200.
curl https://disaster-text-service-19133435506.us-east1.run.app/health
POST /predict
Classify a single piece of text. Returns label (0 or 1) and score (0 to 1).
curl -X POST /predict \\ -H "Content-Type: application/json" \\ -d '{"text":"Forest fire near La Ronge Sask. Canada"}'
POST /predict_batch
Classify up to 100 texts in a single request. Returns an array of predictions.
curl -X POST /predict_batch \\ -H "Content-Type: application/json" \\ -d '{"texts": ["fire", "pizza"]}'
GET /stats
Service statistics: total predictions, average latency, model version, uptime.
curl https://disaster-text-service-19133435506.us-east1.run.app/stats
POST /explain
Model transparency. Returns the top positive and negative TF-IDF features that influenced the prediction.
curl -X POST /explain \\ -H "Content-Type: application/json" \\ -d '{"text":"Forest fire near La Ronge", "top_n": 3}'
